# Instagram Campaign Planner - React Frontend

Build a professional Instagram campaign planning app with drag-and-drop calendar, multi-step campaign creation, and double-sidebar layout. Use ONLY mock data (no backend).

## Tech Stack

```json
{
  "framework": "React 18 + Vite",

  "drag-drop": "@dnd-kit/core, @dnd-kit/sortable",
  "icons": "lucide-react",
  "dates": "date-fns"

```

## Core Features

### 1. Multi-Step Campaign Modal (4 Steps)

**Trigger:** “Create Campaign” button in sidebar

**Step 1 - Purpose:**

- Dropdown: “Product Launch”, “Event Promotion”, “Brand Awareness”, “Engagement Campaign”, “Seasonal Campaign”, “Other”
- Textarea: Campaign description (500 char limit)

**Step 2 - Timeline:**

- Start date picker (min: today)
- End date picker (min: start date + 1)
- Show campaign duration dynamically

**Step 3 - Content Volume:**

- Number input: Posts (default 5, min 1, max 50) with +/- buttons
- Number input: Reels (default 3, min 1, max 30) with +/- buttons
- Show total content pieces

**Step 4 - Review:**

- Textarea: Additional requirements (optional)
- Summary card showing all previous inputs
- “Create Campaign” button

**Modal specs:**

- Max-width 600px, centered, white card, rounded-xl, shadow-2xl
- Progress indicator: 4 circles showing current step
- Navigation: “Back” and “Next” buttons (disabled if required fields empty)
- Close with X button or ESC key
- Validation before proceeding to next step

### 2. Layout - Double Sidebar Design

**LEFT SIDEBAR (240px, fixed, always visible):**

```
┌─ Logo + "Campaign Planner"
├─ "Create Campaign" button (primary blue, full width)
├─ Navigation:
│  • Dashboard (Home icon)
│  • Campaigns (Calendar icon) ← active
│  • Instagram (Instagram icon)
│  • Settings (Settings icon)
└─ User profile card at bottom
```

**CENTER (flex-1, main content):**

- Calendar header: Month/Year with prev/next buttons + “Today” button
- Monthly calendar grid (7 cols, auto rows, min-h-32 per cell)
- Day names header row (Sun-Sat)
- Each cell: white bg, rounded, border, can contain multiple content blocks
- Today highlighted with blue circle on date number
- Weekends: light gray background

**RIGHT SIDEBAR (320px, collapsible):**

- Toggle button (arrow icon) on left edge
- When open: “All Content” header + scrollable list of all content blocks
- Grouped by date with sticky date headers
- Smooth slide animation (300ms)
- Default: Open

### 3. Content Block Design

**Visual (both calendar & sidebar):**

- Border-left: 3px solid (blue for post, purple for reel)
- White bg, rounded-lg, shadow-sm, p-3
- Layout:
  
  ```
  [Drag Handle] [Type Icon] [Title text...]        [Status Badge]
                              Time: 2:00 PM
                              Description...
  ```
- Type icons: Camera (post), Video (reel)
- Status badges: Draft (gray), Scheduled (blue), Published (green)
- Hover: scale 1.02, deeper shadow
- Drag handle: GripVertical icon, gray, cursor grab

**In sidebar only:**

- Add input field for Canva link (placeholder: “Add Canva link…”)
- Show edit (Pencil) and delete (Trash2) icon buttons

### 4. Drag and Drop (CRITICAL)

**Must use @dnd-kit:**

- Any content block is draggable
- Any calendar day cell is a drop zone
- During drag: block at 50% opacity, cursor grabbing
- Drop zone hover: blue-50 bg, dashed blue border
- On drop: update block’s scheduledDate to target day, keep same time
- Within same day: reorderable vertically
- Show toast notification: “Content moved to [date]”
- Smooth animations for all movements

## Design System

**Colors:**

- Primary blue: #3B82F6
- Purple (reels): #A855F7
- Success green: #10B981
- Gray backgrounds: #F9FAFB (page), #FFFFFF (cards)
- Text: #111827 (primary), #6B7280 (secondary)

**Typography:**

- Font: Inter or system font stack
- Headings: font-semibold
- Sizes: text-xs, text-sm, text-base, text-lg, text-2xl

**Spacing:**

- Consistent: 4px, 8px, 16px, 24px, 32px
- Border radius: rounded-lg (8px), rounded-xl (12px)

**Animations:**

- All transitions: 200ms ease-in-out
- Hover: slight scale up + shadow increase
- Active: scale 0.98
- Smooth: use cubic-bezier(0.4, 0, 0.2, 1) for sidebars

## Key Interactions

1. **Create Campaign:** Modal → Generate content blocks distributed evenly across timeline → Show in calendar → Toast: “Campaign created!”
1. **Drag Block:** Pick up → Move to new day → Update scheduledDate → Toast: “Moved to [date]”
1. **Toggle Sidebar:** Smooth slide animation
1. **Navigate Months:** Update currentMonth, re-render calendar
1. **Edit Canva Link:** Click input → Type/paste → Auto-save to state
1. **Delete Block:** Confirm dialog → Remove from state → Toast: “Deleted”

## Toast Notifications

- Position: top-right, fixed
- Style: white, shadow-lg, rounded-lg
- Auto-dismiss: 3 seconds
- Types: Success (green), Error (red), Info (blue)
